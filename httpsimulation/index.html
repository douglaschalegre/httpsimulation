Consulta de HQ's, filmes ou livros

    Como utilizar:
      Primeiro deve ser criado um elemento da classe data da seguinte forma:
    • exemplo = new Data(hq's, filmes, livros)
    • onde hq's, filmes e livros são objetos no formato json
    • como arquivos padrão no sistema foram criados as variaveis hqsJSON, moviesJSON e booksJSON que já contem informações sobre os três porém não é obrigatoria a sua utilização
